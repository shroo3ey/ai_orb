# Orb Pulse Animation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a one-shot "ping" pulse to `AgentOrb` — a button triggers a smooth bell-curve radial expansion that settles back on top of the existing breathing animation.

**Architecture:** A new `uPulse` uniform is added to the shader and summed into the existing radial scale (`p *= 1.0 + breath + uPulse`). The scene layer tracks pulse start time and computes a bell-curve value each frame. React exposes `pulse()` via an imperative handle on `AgentOrb`.

**Tech Stack:** Next.js (breaking-change version — consult `node_modules/next/dist/docs/` if routing/rendering questions come up), React 19 (`forwardRef`, `useImperativeHandle`), Three.js, GLSL.

**Spec reference:** [docs/superpowers/specs/2026-04-18-orb-pulse-design.md](../specs/2026-04-18-orb-pulse-design.md)

**Testing note:** This is a visual WebGL feature. There are no unit tests — verification is done manually on the `/orb` dev page by clicking the Pulse button and observing the animation. Each task ends with the implementer running `npm run dev`, opening the page, and confirming behavior. This is called out explicitly per task.

Using `performance.now() - startTime` keeps pulse timing in the same `elapsed` reference frame the animate loop uses.

- [ ] **Step 4: Compute `uPulse.value` each frame in `animate()`**

In the `animate` function, after the line `material.uniforms.uBreathPhase.value = breathPhase;` (around line 157), add:

```ts
    let pulseVal = 0;
    if (pulseStart !== null) {
      const t = (elapsed - pulseStart) / ORB_CONFIG.pulse.duration;
      if (t >= 1) {
        pulseStart = null;
      } else {
        // Bell curve with ease-out bias: peak slightly before midpoint so the expansion
        // feels snappy and the settle feels unhurried — a "ping" rather than a "swell."
        const eased = Math.sin(Math.PI * Math.pow(t, 0.7));
        pulseVal = eased * ORB_CONFIG.pulse.amplitude;
      }
    }
    material.uniforms.uPulse.value = pulseVal;
```

## Task 5: Expose `pulse` via imperative ref on `AgentOrb`

**Files:**
- Modify: `src/components/AgentOrb/AgentOrb.tsx`

- [ ] **Step 1: Rewrite `AgentOrb.tsx` with `forwardRef` + `useImperativeHandle`**

Open [src/components/AgentOrb/AgentOrb.tsx](../../../src/components/AgentOrb/AgentOrb.tsx). Replace the entire file with:

```tsx
'use client';

import { forwardRef, useEffect, useImperativeHandle, useRef } from 'react';
import { createOrbScene, type OrbState } from './orb-scene';

export type { OrbState };

export interface AgentOrbHandle {
  pulse: () => void;
}

interface AgentOrbProps {
  state: OrbState;
}

export const AgentOrb = forwardRef<AgentOrbHandle, AgentOrbProps>(function AgentOrb(
  { state },
  ref,
) {
  const containerRef = useRef<HTMLDivElement>(null);
  const sceneRef = useRef<ReturnType<typeof createOrbScene> | null>(null);

  useEffect(() => {
    const container = containerRef.current;
    if (!container) return;

    const scene = createOrbScene(container);
    sceneRef.current = scene;

    const handleResize = () => {
      scene.resize(container.clientWidth, container.clientHeight);
    };
    window.addEventListener('resize', handleResize);

    return () => {
      window.removeEventListener('resize', handleResize);
      scene.dispose();
      sceneRef.current = null;
    };
  }, []);

  useEffect(() => {
    sceneRef.current?.setState(state);
  }, [state]);

  useImperativeHandle(ref, () => ({
    pulse: () => {
      sceneRef.current?.pulse();
    },
  }), []);

  return <div ref={containerRef} className="w-full h-full" />;
});
```

**Files:**

- [ ] **Step 1: Wire up the ref and button**

Open [src/app/orb/page.tsx](../../../src/app/orb/page.tsx). Replace the entire file with:

```tsx
'use client';

import { useRef, useState } from 'react';
import {
  AgentOrb,
  type AgentOrbHandle,
  type OrbState,
} from '@/components/AgentOrb/AgentOrb';

const STATES: OrbState[] = ['idle', 'conscious', 'subconscious'];

export default function OrbDemoPage() {
  const [state, setState] = useState<OrbState>('idle');
  const orbRef = useRef<AgentOrbHandle>(null);

  return (
    <main className="fixed inset-0 bg-black">
      <AgentOrb ref={orbRef} state={state} />
      <div className="absolute bottom-10 left-0 right-0 flex justify-center gap-8">
        {STATES.map((s) => (
          <button
            key={s}
            onClick={() => setState(s)}
            className={`font-mono text-sm tracking-wide transition-opacity ${
              state === s
                ? 'text-white underline underline-offset-4'
                : 'text-white/40 hover:text-white/70'
            }`}
          >
            {s}
          </button>
        ))}
        <button
          onClick={() => orbRef.current?.pulse()}
          className="font-mono text-sm tracking-wide text-white/40 hover:text-white/70 transition-opacity"
        >
          pulse
        </button>
      </div>
    </main>
  );
}
```
